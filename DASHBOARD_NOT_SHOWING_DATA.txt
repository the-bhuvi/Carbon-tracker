# Dashboard Not Showing Your Data? Here's the Fix

## The Problem
You added data to the system but it's not appearing on the dashboard.

## The Solution (30 seconds)
1. **Go to this URL:** `http://localhost:5173/refresh-dashboard`
2. **Click the button:** "Refresh Current Month"
3. **Wait for:** Green "Success" message
4. **Go back to:** Dashboard page
5. **See your data!** ‚úì

That's it! Your data should now be visible.

## Why Did This Happen?

The dashboard displays data from "summary" tables in the database. Think of it like this:

```
Your Data (monthly_audit_data)
    ‚Üì
    ‚Üì (needs refresh to calculate)
    ‚Üì
Summary Tables (monthly_summary, academic_year_summary)
    ‚Üì
    ‚Üì (dashboard reads from these)
    ‚Üì
Dashboard Display
```

When you use the **Admin Input form**, this refresh happens automatically. When you add data directly to the database, it doesn't. That's why you don't see it.

## Common Questions

**Q: Will I need to do this every time?**
A: Only if you add data directly to the database. If you use the Admin Input form, it's automatic.

**Q: The page says "Refresh Current Year" - should I use that?**
A: Only if you added data for multiple months. For single months, use "Refresh Current Month".

**Q: I clicked refresh but still don't see data?**
A: Make sure to:
1. Check that the Dashboard year/month matches your data
2. Refresh your browser page (press Ctrl+R)
3. Make sure you're looking at the right view (Monthly vs Academic Year)

**Q: Can I add data directly to the database?**
A: Yes, but then you need to refresh. It's easier to use the Admin Input form.

**Q: The /refresh-dashboard page won't load?**
A: Make sure you're logged in as an admin, not as a student.

- [ ] Are you looking at the right month/year?
